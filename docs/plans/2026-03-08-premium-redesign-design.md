# Дизайн-документ: Премиальное обновление в стиле TimeTracker 💎

## 1. Обзор
Цель — привести дизайн ZenWhisper к эстетике «Premium Dark», вдохновленной [TimeTracker Dashboard](https://timetracker.assistant-n8n.top/dashboard). Дизайн характеризуется глубокими темными тонами (Slate-900), неоновыми синими акцентами (Blue-500) и эффектом матового стекла (Glassmorphism).

## 2. Дизайн-система (Token Set)

### 🎨 Цвета (Colors)
| Имя | Код (HEX) | Применение |
|-----|-----------|------------|
| **Background** | `#0f172a` | Основной фон окон |
| **Surface** | `rgba(30, 41, 59, 0.4)` | Фон карточек (Glassmorphism) |
| **Primary** | `#3b82f6` | Акценты, активные кнопки, иконки |
| **Accent Glow** | `rgba(59, 130, 246, 0.2)` | Внешнее свечение (Drop Shadow) |
| **Success** | `#10b981` | Статус "Recording", "Done" |
| **Danger** | `#ef4444` | Кнопка удаления, ошибки |
| **Text Primary** | `#f8fafc` | Основной текст заголовков |
| **Text Muted** | `#94a3b8` | Вспомогательный текст |

### 🧱 Компоненты (Components)

#### 💳 Карточки (`#PremiumCard`)
- **Скругление:** 12px.
- **Граница:** `1px solid rgba(148, 163, 184, 0.1)`.
- **Эффект:** Фон с прозрачностью + легкий эффект свечения при наведении.

#### 🔘 Кнопки (Buttons)
- **Primary:** Градиент от `#3b82f6` к `#2563eb`, скругление 10px-12px.
- **Hover:** Увеличение (scale 1.02), усиление яркости фона.
- **Active:** Легкое сжатие (scale 0.98), изменение тени.

#### 🗄️ Боковая панель (Sidebar)
- **Фон:** `#0f172a` (чуть темнее основного или с градиентом).
- **Активный пункт:** Левая граница 3px (`#3b82f6`) и неоновое свечение за текстом.

## 3. Архитектура обновления
1.  **`src/zenwhisper/ui/styles.py` [NEW]**: Центральный модуль со всеми QSS-шаблонами и цветовыми константами.
2.  **`src/zenwhisper/ui/base.py`**: Обновление базового класса `HubPage` для автоматического применения стилей.
3.  **Refactoring Components**: Поэтапное обновление `hub.py`, `analyst.py`, `windows.py`.
