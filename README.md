# ZenWhisper

**ZenWhisper** — это премиальное приложение для голосовой диктовки в Linux, созданное для тех, кто ценит приватность, эстетику и скорость.

## Основные особенности
- **100% Локально**: Транскрипция и обработка текста происходят только на вашем компьютере.
- **Эстетика Zen**: Плавающая неоновая звуковая волна, реагирующая на ваш голос.
- **Умная очистка**: Локальная LLM удаляет слова-паразиты и исправляет пунктуацию.
- **Премиальный звук**: Приятные звуковые отклики при работе.

## Установка (в разработке)
Для запуска из исходников:
```bash
# Установите зависимости
uv sync
# Запустите приложение
uv run zenwhisper
```

## Технологии
- **UI**: PyQt6
- **STT**: faster-whisper
- **Audio**: sounddevice, pydub
- **Hotkeys**: pynput
